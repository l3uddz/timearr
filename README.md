# timearr

Simple Custom Import Script to reset the mod-time of files imported by Sonarr/Radarr/Lidarr.

# Setup

- Create new Connect entry in Sonarr/Radarr/Lidarr
- Set path of the binary
- Ensure Import & Upgrade are enabled
- Save
